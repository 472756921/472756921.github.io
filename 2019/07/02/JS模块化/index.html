CommonJS

服务器端js模块化的规范，同步的模块加载–(代表：NodeJS)

AMD

浏览器端js模块化的规范，提前执行（异步加载：依赖先执行）–(代表：RequireJS)

CMD

也是浏览器端js模块化的规范，延迟执行（运行到需加载，根据顺序执行）–(代表：SeaJS)

ES6模块化

ES6模块化出现后，将逐步地取代commonJS和AMD，成为浏览器和服务器的通用模块化解决方案。
但是ES6模块无法直接运行在浏览器环境，需要转成ES5后才能正常运行。
